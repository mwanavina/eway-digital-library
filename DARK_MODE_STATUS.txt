================================================================================
TECHNICAL IMPLEMENTATION
================================================================================

ARCHITECTURE:
  ThemeProvider
    ├─ Detects system preference
    ├─ Reads/writes localStorage
    ├─ Manages "dark" class on <html>
    └─ Exposes __toggleTheme to window

  ThemeSwitcher
    ├─ Client component
    ├─ Reads theme from localStorage
    ├─ Shows appropriate icon
    └─ Toggles on click

  CSS Variables
    ├─ Defined in app/globals.css
    ├─ OKLch color space (perceptually uniform)
    ├─ 25+ variables for light mode
    └─ 25+ variables for dark mode

DATA FLOW:
  Browser Storage (localStorage)
         ↓
    ThemeSwitcher (reads/writes)
         ↓
    HTML Element (dark class)
         ↓
    CSS Variables (updated)
         ↓
    UI Colors (all change)
